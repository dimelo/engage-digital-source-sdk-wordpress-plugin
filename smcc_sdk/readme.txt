The simplest way to integrate your WordPress blog with RingCentral Engage Digital.

== Description ==

The RingCentral Engage Digital Source SDK is an easy way for any content provider to integrate his data into our application: RingCentral Engage Digital.
Agents and community managers using Engage Digital will be able to interact in an efficient and trackable way with the
content provider and benefit from all Engage Digital features (autocategorization, knowledge database, reporting, ...).

If you are interested in developing a client for our SDK and you want to have a method to test your implementation,
you will first need to speak to us. This way we'll be able to provide you with an Engage Digial test domain.

Head over to our product page to get more details about what we do: https://www.ringcentral.fr/digital-customer-engagement.html

== Installation ==

The simplest way to install the plugin is by uploading the zip archive:

1. Go to Plugins > Add new
1. Click on Upload
1. Choose the zip archive
1. Click Install now

Alternatively you can install it manually:

1. Upload the `smcc_sdk` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

Please remember to configure the plugin after you install it.

Note: *The option `Comment author must fill out name and e-mail` must be enabled.*
